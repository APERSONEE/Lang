About the NWVBUG Lang Project

The Lang Project is a project headed by NWVBUG Development of Lye Software.

What is Lang?

Lang is not just a term and defintion study program as it is customized to be highly useful in language classes, from which the name is derived. Lang is extremely user customizable, from the user interface to the way the study algorithm is handled. Lang supports cloudsaving your studysheets, and as with all Lye Software products it uses the proprietary and secure Lye Account system to do so. Lang Studysheets are saved in the proprietary .lang format to ensure that you can securely save your Studysheets to a local machine.

Lang excels even in non-language courses due to its simplicity in UI design, as it keeps the distractions away while you work.
