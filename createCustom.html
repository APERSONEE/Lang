You can make custom verb sheets using either the built in sheet creator or do it manually.

Manual:

Create a .txt file (name does not matter).

input the verbs you want in this syntax:

mutli verbs

[infinitive form, je, tu, il, nous, vous, ils]

for example, [parler, parle, parles, parle, parlons, parlez, parlent]

single verbs

[infinitive form, stem]

for example, [être, avoir été]
