Changelog
Lang has been updated from Lang 3.0 to Lang 3.1. The main hallmarks of this update is the addition of asynchronous pings to the server, in place of the previous synchronous ones. This improves user experience greatly and allows for more versatile features.

Various UI changes have been made, but we still are working on the homepage (when not logged in) and are deciding what we want the studying pages to look like. Thank you for supporting Lang.

Make sure to follow This GitHub Repository if you want to be notified as soon as Lang is updated.
