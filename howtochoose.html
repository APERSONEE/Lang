What is the difference between Single and Multi?

For Language Classes

Single Answer Sheets have a single stem or one way to conjugate it. These verbs typically appear in the passe composee and the conditionnel.

Multi Answer Sheets have a different conjugation for each pronoun. These verbs are typically in the present tense.

For other classes

It is extremely unlikely that you will have to use Multi verbs for anything other than language classes.

Single is the same as Term/Definition. This is what should be used for typical question/answer flashcard type activities.
